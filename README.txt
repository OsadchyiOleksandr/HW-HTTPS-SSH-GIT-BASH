В проекті має бути реалізовано функціонал конвертації миль в кілометри, кілометрів в милі. Розробку ведіть під Git та GitHub (достатньо однієї гілки master), використовуючи Git Bash/Terminal та SSH. Зверніть увагу на створення комітів, формування повідомлень комітів.

(1) Сгенеруйте та налаштуйте SSH ключі.

(2) Початково створіть проект Converter на локальній машині через IntelliJ IDEA (IDE).

(3) Додайте функціонал виведення інформації про проект в консоль.

(4) Зробіть початковий коміт.

(5) Створіть репозиторій на GitHub.

(6) Налаштуйте доступ до репозиторію на GitHub через SSH.

(7) Запуште зміни в проекті на GitHub.

(8) Додайте функціонал конвертації миль в кілометри.

(9) Зробіть коміт цих змін.

(10) Запуште ці зміни на GitHub.

(11) Додайте функціонал конвертації кілометрів в милі.

(12) Зробіть коміт цих змін.

(13) Запуште ці зміни на GitHub.

(14) Посилання на свій GitHub репозиторій зазначте в LMS.
